# Security Automation Toolkit

A hybrid CLI + API + Dashboard tool for automating key cybersecurity tasks.

## Features
- Automated Port & Service Scanner
- Log File Analyzer
- Threat Intelligence Lookup
- Vulnerability Checker
- PDF + JSON Report Generator
- FastAPI Backend
- React Dashboard
- Typer CLI

## Tech Stack
Backend: Python + FastAPI
CLI: Python + Typer
Dashboard: React + Vite + Tailwind
Database: SQLite (simple & clean; upgradeable later)
Reports: pdfplumber + reportlab
Threat APIs: AbuseIPDB + OTX + NVD + Vulners

## To run the dashboard
```
cd dashboard
npm run dev
```
It will again give:
```
VITE v7.x.x ready
Local: http://localhost:5173/
```
